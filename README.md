# 📌 Language Detection using Naive Bayes

## 📖 Description
This project builds a text classification system to detect the language of a given text using Natural Language Processing (NLP) techniques. It uses the Naive Bayes algorithm, a simple yet effective probabilistic classifier, to identify the language based on textual patterns.

The model is trained on labeled text data from multiple languages and predicts the language of new input text accurately.

## 🎯 Objective
- Automatically detect the language of input text
- Build a fast and efficient classification model
- Apply NLP techniques for text processing

## 🛠️ Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- NLTK / SpaCy

## ⚙️ Features
- Detects multiple languages
- Fast prediction using Naive Bayes
- Handles different text inputs
- Easy to extend with more languages
